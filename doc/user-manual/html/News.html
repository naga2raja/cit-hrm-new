News

Click “News”, To add news which will be shown on dashboard. Based on the selection of Project it will be displayed to Employees. If selected Project, then it will be shown for Admin and Project employees only. If Project is selected as ALL, then it will be shown for all employees.

Adding Sample news, Project as ALL >> Click on “ADD” button to add new News. Enter Title, Details, select Category, select Project, select Status and click on “Save” button to add news to dashboard. If the Status is “in-active” then it won’t be visible on dashboard.

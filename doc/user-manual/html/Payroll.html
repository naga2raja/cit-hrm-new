Payroll

In Payroll Module, Admin will upload all employee Payslips and employees can download own Payslips. Note: PDF file name must in below format, <<Employee ID>>-<<Month>>-<<Year>>.pdf e.g., 12000000-Jan-2021.pdf Click on “Payroll”, to view previously added payroll data.

Click on “Add” button to add new Pay slip. Enter Employee Name, select Month, Upload Pay slip, write comment if any and click on “Save” button to add.
